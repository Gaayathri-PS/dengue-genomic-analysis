# Dengue Virus Genomic Analysis

**Author:** P.S. Gaayathri
**Date:** June 2026  
**Institution:** Vellore Institute of Technology,Vellore, B.Tech Biotechnology — Year 1  

## Project Overview
Comparative genomic analysis of all four dengue virus serotypes 
(DENV-1 to DENV-4) using publicly available genome sequences from NCBI.

## Objectives
- Download and process complete dengue genome sequences from NCBI
- Perform multiple sequence alignment across all four serotypes
- Construct a phylogenetic tree to visualise genomic divergence
- Compare GC content across serotypes using Python

## Tools Used
- NCBI GenBank (data source)
- MEGA 11 (alignment and phylogenetic tree)
- Python 3 with BioPython, pandas, matplotlib (analysis)
- Jupyter Notebook (coding environment)

## Key Question
How genomically distinct are the four dengue serotypes, and what 
does this tell us about cross-serotype immunity failure?
